# toy_robot_puzzle

### Build With

* [.NET Core](https://dotnet.microsoft.com/en-us/download)

### Prerequisites

* Install latest [.NET](https://dotnet.microsoft.com/en-us/download)

## Getting started

1. Clone the repo
   ```sh
   git clone https://github.com/angni01/toy_robot_puzzle.git
   ```
2. Building the app
   ```sh
   dotnet build
   ```
2. Running the app
   ```sh
   dotnet run
   ```
4. Running unit test
   ```sh
   dotnet test
   ```
